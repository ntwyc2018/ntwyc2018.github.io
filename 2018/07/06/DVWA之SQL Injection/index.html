SQL Injection

SQL Injection（SQL注入）是指攻击者通过注入恶意的SQL命令，破坏SQL查询语句的结构，从而达到执行恶意SQL语句的目的。

SQL注入步骤

  • 判断是否存在注入，注入是字符型还是数字型
  • 判断SQL查询语句中的字段数
  • 爆出当前数据库
  • 爆出数据库中的表
  • 爆出表中的字段名
  • 爆出该表的字段名的内容
